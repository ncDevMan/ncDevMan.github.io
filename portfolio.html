Web Applications

Aloha Music Hall

This project was a small scale, ticket selling web application. This web application uses Bootstrap (HTML & CSS) and JavaScript on the front-end. The back-end of this application uses PHP and a MySQL database.

Home Page

This is the home screen what comes up when you first come to the website before logging in or registering as a customer or member.

View Non-Logged In Events

When the View events page pulls up it will look similar to the picture above and will have a list of event with their times and names and price. If you are not logged in it will take you to a page without the button

Login

To log in you must first click the menu button at the top of the screen that says “Login”. In order to complete login you need to enter your user ID and your password. If you haven’t registered yet, you can either press the “Not a Member?” button or the Register New Member menu button

Register Customer

This is the form that you use to register a new member. You will enter the information needed and then decide whether you want to be a member or not by checking the box above the submit button. After you are done with the form you can click the submit button.

User Creation

After you press the submit button, if done correctly, a page similar to the picture above will pull up and it will give ou your userID and will automatically log you in. From here you can either press the Aloha music Hall home button or go on to reserve a seat. If you need to you can change your personal information in the “Change Personal Information” Menu button which will be explained later.

Reserve a Seat

To reserve seating you must be logged in and then press either the menu button at the top of the screen or press the reserve now button in the “View upcoming events” page. The page to reserve seating looks like the picture below.

Cancel Reservation

When you get to the Cancel Reservation page, it will list the event that you have reservations for along with their ids, time, and available seats. To cancel you must put the appropriate ID in the form box and then put login credentials for security verification and press cancel.

Edit or Change Information

To change personal information you must first be logged in or the menu option will not show. You then need to press the “Change Personal info” menu item. And then a form will show up similar to the picture below.

Aloha Inventory System

This project was an inventory system as a web application. This web application uses HTML and JavaScript on the front-end. The back-end of this application uses C#(ASP.NET) and a MySQL database.

Landing Page

You will land here as an Aloha associate or Amazon employee. You will then be able to access the system. There are 2 Buttons at the top labeled "Place an Order" and "View Results".

Place An Order

Here is where you can place a new order for a customer. Also, above the Customer Form you will see your Guitar inventory. On your right column you will see your part inventory- followed by your supplier inventory.

Edit or Change Information

After you have placed an order, or clicked on the Results button- the sales analysis will appear. Here you can see all of your orders for the past CURRENT month. On the right column you can see your commission, after the 2% has been added to the total profit this only shows the CURRENT month. Following the Commission you can see how many totals parts have been used for this current month.

NBA Shot Analysis

This project's intent was to create data analysis from certain player's shooting. This project was implemented in Python and parsed a JSON file to handle this data. The Python Libraries implemented were Pandas, MatPlotLib, Seaborn and IPython Display. The script then outputs different information into the command prompt.

There is still lots I want to do with the project. This data is from the 2014/2015 season. I would like to grab this season's information, save it all in a database and then crunch some greater numbers!
